# 运行说明

## 本地测试

安装依赖：

```bash
pip install -r requirements.txt
```

验证样例数据：

```bash
python3 00-自动化/validate_opportunities.py 04-模板/30条样例机会.csv
```

## 生成即时机会提醒

```bash
python3 00-自动化/collect_opportunities.py --mode instant
```

如果发现新机会，会写入：

- `04-数据库/机会数据库.csv`
- `06-邮件输出/YYYY-MM-DD-机会提醒.html`，仅作为极简邮件摘要备用
- `05-历史记录/已提醒链接.csv`

## 生成网页仪表盘

```bash
python3 00-自动化/generate_dashboard.py
```

输出：

- `docs/index.html`

## 生成每周汇总

```bash
python3 00-自动化/collect_opportunities.py --mode weekly
```

## 发送邮件

```bash
python3 00-自动化/send_email.py 06-邮件输出/生成的文件.html
```

## GitHub Actions

`.github/workflows/opportunity-radar.yml` 默认：

- 每 6 小时监测一次并提醒新机会。
- 每次运行后生成网页仪表盘。
- 邮件默认不发送长列表；手动运行时可选择发送极简提醒。
- 每周一北京时间 09:00 生成周汇总。
- 支持手动运行。
